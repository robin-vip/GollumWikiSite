# apache mysql php
操作系统： ubuntu 20.04

## LAMP 安装
1. LAMP 的意义  
`L: linux`, `A: apache`, `M: mysql`, `P: php`

2. LAMP 安装  
* sudo apt-get install tasksel
* sudo tasksel install lamp-server
