# office word booknotes

1. 横版
* 全篇  
布局 -> 纸张方向 -> 横向  
* 本节  
布局 -> 页面设置 -> 纸张方向(横向) -> 应用于(本节) -> 确定
* 指定范围
```
a. 将光标移动到设置的开始处；
b. 布局 -> 页面设置 -> 纸张方向(横向) -> 应用于(插入点之后) -> 确定
c. 将光标移动到设置的结束处；
d. 布局 -> 页面设置 -> 纸张方向(纵向) -> 应用于(插入点之后) -> 确定
```
